This release upgrades the database to version 153, introduces employer-borne SOCSO Invalidity contribution support, and fixes payslip and time attendance issues. (*242*)

<!-- truncate -->

## Database

Database upgraded to version **153**

---

## Features

- Added option for employer to bear only the employee **SOCSO Invalidity** contribution (#6325)
- Added batch edit support for all **Sync Time Attendance** fields
- Added **Maintain User** to Web
- Added **Maintain Document Number** to Web
- The post date of **Advance Deduct** is now assigned based on the employee's most recent payroll cut-off day before the month; otherwise, it defaults to the end of the month (#5820)

---

## Updates

- Added **Attendance Reward** posting to **Deduction** support
- Revised time attendance report to exclude resigned employees from the license query
- Removed Action Code when Post Date is null during payroll update
- Added **Request Status** column in **Sync Cloud** > **Pending Claim**

---

## Bug Fixes

- Fixed **PR.SOCSO.Borang8A** report not displaying fraction amounts less than 0.09 correctly
- Fixed Clock Status showing **Absent** when employee had an approved leave applied
- Fixed **Payslip** report **Total Rest Day** and **Total Public Holiday** not including Rest Day Special Rate and Public Holiday Special Rate (#6326)
- Fixed invalid prompt edit value when filtering incomplete NRIC in **Maintain Employee** (#6330)
