This release upgrades the database to version 151, introduces comprehensive SOCSO 2026 support, and resolves PCB calculation, bank file, and paysheet clipboard issues. (*240*)

<!-- truncate -->

## Database

Database upgraded to version **151**

---

## Features

- Added **SOCSO 2026** support with new calculator, data provider, paysheet form, OPF, payroll processor, and web app paysheet integration
- Added **PR.SOCSO.Report.2026** report format
- Added **Hong Leong Connect Pro** excel bank format (#6310)
- Added **Resign Date** and **Contract Expiry Date** to **Maintain Employee History** (#6316)

---

## Bug Fixes

- Fixed **PCB** normal not calculated when a pending transaction contributing to PCB normal is processed via **Ad Hoc** payroll (#6273)
- Fixed **Beneficiary Citizenship** and **Citizenship Country** columns not populated with empty values for detail records in **Maybank2E-RC Universal Payment** file format (#6236)
- Fixed `PRKey` value not excluded when pasting records from clipboard into paysheet detail dataset (#6313)
