This release adds support for PCB 2026, new TP1 and TP3 reports for year 2026, includes the AmAccess AmPayment Excel 80F file format, and addresses various bug fixes and updates. (*236*)

<!-- truncate -->

## Features

- Added **AmAccess AmPayment Excel 80F** file format; renamed **e-AmBiz Bulk Payment** to **AmPayment** (#6184)
- Added **TP1** and **TP3** reports for year 2026
- Added **Print PCB 2026**

## Updates

- Renamed **Employment Contract Stamp Duty** to **Contract Stamp Duty**
- Revised description of optional tax relief items for year 2026
- Reordered **PCB Calculator** print screen section order
- Adjusted **Payroll Calculator** form width

## Bug Fixes

- Fixed **Income tax CP8D report** "Tax exempt" column not excluding tax exempt compensation and gratuity (#6249)
- Fixed **Stamp Duty XML** error "Company state code not found" when company state is "Pulau Pinang" (#6250)
- Fixed **Stamp Duty XML** not being encoded in UTF-8
- Fixed **Yearly Payroll Report** prompt error "Dynamic SQL Error alias C conflicts with an alias in the same statement" (#6247)
- Fixed issue where **SQLPay.Misc.DP** is loaded on application startup
- Fixed **Dashboard** prompt error "SQLPay.Dashboard.GUI.bpl not exist" upon clicking (#6244)
