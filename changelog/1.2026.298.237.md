This release introduces Public Holiday Special Rates, WebApp integration, E-Claim email notifications, and includes significant improvements to TMS and Sync Cloud processing logic. (*237*)

<!-- truncate -->

## Database

Database upgraded to version **148**

## Features

- Added **Mobile Connect** support
- Added **Public Holiday Special Rate 1** and **2**
- Added **Send EClaim Email Notification** option in **Company Profile | E HRMS**
- Added ability to view script in **Sync Cloud**

## Updates

- Removed **Analyse Data Integrity** access right
- Revised **TMS** processor to calculate all **Before Work** and **After Work OT**
- Added **OEM** info to **SQLPaySVC** project
- Revised **TA.Cloud** implementation

## Bug Fixes

- Fixed **AmPayment** invalid bank code in IFT record (#6251)
- Fixed **TMS** wrong calculation for multiple in/out logs
- Fixed **TMS** issue where extra empty log caused **Summary** to be "Absent"
- Fixed **TMS** leave records having script but no leave rules assigned
- Fixed **TMS** issue with no log on **Rest Day** but having summary record
- Fixed **Sync Cloud** syncing null logs to **EMS**
- Fixed **Sync Cloud** missing **Absent** leave record
- Fixed **Sync Cloud** syncing incomplete logs after 9am
