This release addresses multiple bugs including EPF payment file errors, incorrect TMS summary values, and issues with the 2026 payroll calculator. (*237*)

<!-- truncate -->

## Bug Fixes

- Fixed incorrect summary and transaction values:
  - Duplicate OT records
  - **Absent** record showing when log is **OnTime**
  - Summary values not updating after log synced after 9am
- Fixed **Clock In** `IsOT` field not setting to False when uploaded to server
- Fixed inability to display special characters on grid
- Fixed "Line 1: Invalid format" error when submitting **EPF payment file** to **CIMB Bank** (#6279)
- Fixed "Invalid file format length" error when submitting **EPF payment file** in **PB Enterprise** text format to **Public Bank** (#6277)
- Fixed "Access Violation" error in **Payroll Calculator 2026** when clicking Calculate
- Fixed "Interface not supported" error when saving paysheet after adding **Employment Loss Compensation** record (#6274)
- Fixed **Paysheet Claim** details allowing an empty claim code
