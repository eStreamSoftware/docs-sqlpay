This release upgrades the database to version 152, adds Non-Employment Injury Scheme support for employees with multiple employers, and fixes bugs related to leave deletion rights, Hong Leong bank format, and Paysheet PCB amount. (*241*)

<!-- truncate -->

## Database

Database upgraded to version **152**

---

## Features

- Added **Non-Employment Injury Scheme** field to **Maintain Employee**, allowing employees with multiple employers to contribute through only one employer (#6323)

---

## Bug Fixes

- Fixed leave application records outside the acceptable transaction date range being deletable by users without required rights (#6108)
- Fixed **Beneficiary Account** not prefixed with "D" for non-HLBB beneficiary banks in **Hong Leong Bank Connect Pro** format (#6322)
- Fixed **Paysheet** prompt error "Field 'PCBAmount' not found" in Mobile Payroll (#6321)
